# 🎮 Python Mini Games

Welcome to my Python Mini Games repository!

This repository contains beginner-friendly Python projects that I built to practice programming concepts.

## 📂 Projects

### 1. Number Guessing Game
- The computer generates a random number between 1 and 100.
- The player keeps guessing until the correct number is found.
- Uses the `random.randint()` function.

### 2. Word Guessing Game
- The computer randomly selects a fruit name.
- The player tries to guess the correct word.
- Uses the `random.choice()` function.

## 🛠️ Concepts Used

- Python Basics
- Variables
- Input and Output
- if-else Statements
- Loops
- Random Module
- `randint()`
- `choice()`
